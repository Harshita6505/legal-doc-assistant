# Legal Doc Assistant

Legal Doc Assistant is an AI-powered application designed to help users analyze, summarize, and interact with legal documents easily. It uses large language models (LLMs) and intelligent search techniques to provide useful summaries, insights, and answers to questions based on uploaded legal documents.
